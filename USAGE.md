# Usage Guide

## Basic Search

1. Open http://localhost:3000
2. Enter your search query
3. Select search mode (Aggregation/Crawling/Hybrid)
4. Choose target languages
5. Select sources (DuckDuckGo live, DeepWeb/DarkWeb soon)
6. Click "Cerca" (Search)

## Search Modes

### Aggregation Mode
- Fastest option
- Queries existing search engines
- Best for: Quick lookups, general queries

### Crawling Mode
- Thorough option
- Active web scraping
- Best for: Deep research, specialized content

### Hybrid Mode
- Combines both methods
- Maximum coverage
- Best for: Comprehensive research

## Sources

### DuckDuckGo (Clearnet)
Live results fetched directly from DuckDuckGo.

### Clearnet
Traditional web (Google, Bing, etc.) — roadmap

### Deep Web
Non-indexed content (forums, databases, academic repositories) — roadmap

### Dark Web
Tor network (.onion sites) — roadmap

## Tips

- Use specific keywords for better results
- Select relevant languages for your search
- Aggregation mode is faster for simple queries
- Use Hybrid mode for comprehensive research
